## 📖 About The Project

**Al-Falah Foundation** is a comprehensive community welfare platform designed to connect the people of Kotli Pathanan, showcase village progress, and serve humanity. This digital hub brings together essential Islamic tools, community management features, and transparent administration in one unified interface.

Our mission is to bridge hearts across borders—whether geographic or generational—by providing accessible Islamic resources, community engagement tools, and transparent management of welfare initiatives.

### 🌍 Community Impact
- **500+** Community members served
- **6** Daily prayer time displays
- **30** Days of Ramadan calendar
- **Real-time** Qibla direction finder
- **Transparent** Income/Expense tracking

## ✨ Key Features

### 🏠 **Main Dashboard** (`index.html`)
- **Multi-Language Support** - Seamless toggle between English and Urdu with full RTL support
- **Live News Ticker** - Scrolling announcements with dynamic content
- **Current Prayer Display** - Real-time prayer tracking with next prayer countdown
- **Mobile Responsive Navigation** - Adaptive menu for all screen sizes
- **Dynamic Member Management** - Add, search, and display community members
- **Admin Panel** - Secure income/expense tracking with password protection (`admin5`)
- **Gallery System** - Upload and display community images
- **Vision Board** - Track community projects with status indicators
- **Donation Integration** - JazzCash and bank transfer details

### 🕌 **Prayer Board** (`prayer-board.html`)
- **Full Prayer Schedule** - All 6 daily prayers with Azan and Iqama times
- **Dynamic Clock** - Live time display with seconds
- **Hadith of the Day** - Rotating collection of Islamic sayings
- **Prayer Phase Alerts** - "Close your mobile" reminders during prayer
- **Post-Prayer Azkar** - Fullscreen dhikr overlay with countdown timer
- **Customizable Settings** - Manual time override, iqama offsets, content management
- **Persian/Arabic Support** - Full RTL and Arabic script rendering

### 🛠️ **Islamic Tools** (`tools.html`)
- **Complete Prayer Schedule** - All prayers with visual active highlighting
- **Qibla Direction Finder** - GPS-based compass with precise direction
- **Zakat Calculator** - Automated 2.5% calculation on assets
- **Ramadan Calendar** - Pre-generated 2026 timetable with Sehri/Iftar times
- **Resource Links** - Quick access to Quran.com, Sunnah.com, and Aladhan API

## 🛠️ Tech Stack

### Core Technologies
- **HTML5** - Semantic markup with accessibility features
- **CSS3** - Custom styling with animations and responsive design
- **JavaScript (Vanilla)** - Native JS with localStorage persistence

### Frameworks & Libraries
- **Tailwind CSS** - Utility-first styling framework
- **Font Awesome 6** - Comprehensive icon library
- **Google Fonts** - Amiri, Noto Nastaliq Urdu, Poppins

### APIs & External Services
- **Aladhan API** - Prayer times and Qibla direction
- **Geolocation API** - Device location for Qibla calculation

### Persistence
- **localStorage** - Client-side data storage for:
  - Member records
  - Transaction history
  - Gallery images
  - Project plans
  - Prayer time cache
  - User preferences

## 🚀 Quick Local Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- VS Code (Recommended)
- Live Server extension (VS Code)

### Installation Steps

1. **Clone the Repository**
```bash
git clone https://github.com/yourusername/al-falah-foundation.git
cd al-falah-foundation
```

2. **Open in VS Code**
```bash
code .
```

3. **Launch with Live Server**
- Right-click on `index.html`
- Select "Open with Live Server"
- Or use the Live Server extension button in VS Code

4. **Alternative Method**
- Simply double-click any HTML file to open in your browser
- All features work locally without a server

### ⚡ Quick Access Points
| Feature | File | Key Function |
|---------|------|--------------|
| Main Dashboard | `index.html` | Home page with all features |
| Prayer Board | `prayer-board.html` | Dedicated prayer display |
| Islamic Tools | `tools.html` | Calculator and resources |
| Admin Panel | `index.html` → Admin Panel | Password: `admin5` |
